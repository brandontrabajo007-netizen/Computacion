Conclusión sobre CSS

CSS es clave para transformar una estructura simple en una interfaz profesional. Un buen resultado depende de mantener consistencia en espaciado, contraste y alineación.

Reflexión sobre la especificidad en CSS

La especificidad permite controlar con precisión qué reglas se aplican. Dominarla evita conflictos entre selectores y hace más mantenible el proyecto.

Selectores avanzados que más me costaron

  • Selector descendiente: me costó porque requiere leer bien el árbol del documento antes de escribir el selector.
  • Pseudo-clase: me costó comprender cuándo conviene usar :hover y cuándo usar otros estados de accesibilidad.
  • Pseudo-elemento: me costó visualizar su uso inicial, pero luego entendí que es útil para detalles decorativos sin tocar el HTML.

Mobile-First en este proyecto

Mobile-First significa empezar por la vista móvil para asegurar que lo esencial sea claro y usable. Después se agregan mejoras para pantallas grandes. En este trabajo se aplicó construyendo una base vertical y luego activando Grid desde 768px.
