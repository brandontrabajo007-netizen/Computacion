# Ingeniería Web I – Guía de Laboratorio

## Descripción del Proyecto

Este proyecto corresponde a la práctica de laboratorio de **Ingeniería Web I**, enfocada en el uso de **CSS avanzado, Flexbox, CSS Grid y diseño responsivo con metodología Mobile-First**.

El objetivo es recrear un **componente de perfil de una red social**, aplicando selectores avanzados de CSS sin el uso de clases ni IDs adicionales, además de desarrollar un **blog personal de aprendizaje** por cada integrante del grupo.

# Funcionalidades Implementadas

* Diseño de perfil de red social.
* Uso de **CSS Grid** para el layout principal.
* Uso de **Flexbox** para la navegación del perfil.

* Implementación de **diseño responsivo con Mobile-First**.
* Galería de imágenes con efecto **hover**.

---

# Metodología Mobile-First

El desarrollo se realizó siguiendo el enfoque **Mobile-First**, lo que significa que primero se diseñó la interfaz para dispositivos móviles y posteriormente se agregaron estilos para pantallas más grandes mediante **media queries**.

Beneficios del enfoque Mobile-First:

* Mejora la experiencia en dispositivos móviles.
* Optimiza el rendimiento de la página.
* Facilita la escalabilidad del diseño.
* Permite adaptar el layout fácilmente a diferentes resoluciones.
